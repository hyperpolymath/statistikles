= statistease — Quick Start for Platform Maintainers
:toc:
:toclevels: 2

== Overview

This guide covers packaging, deploying, and maintaining statistease for
distribution on your platform.

== Runtime Dependencies

* Julia 1.10+ (package dependencies are resolved by `Pkg` from `Project.toml`:
  CSV, DataFrames, Distributions, HTTP, JSON3, StatsBase)
* Optional, for the conversational interface only: an OpenAI-compatible LLM
  endpoint (defaults to LM Studio at `http://localhost:1234/v1`; override with
  `STATISTEASE_LM_URL`, `STATISTEASE_API_KEY`, `STATISTEASE_MODEL`)

== Build from Source

[source,bash]
----
git clone https://github.com/hyperpolymath/statistease.git
cd statistease
just build-release
----

Output: a precompiled Julia project environment (no standalone binary — the
package is loaded with `julia --project=. -e 'using StatistEase'`)

== Packaging

=== Guix

[source,bash]
----
guix build -f guix.scm
----

=== Nix

[source,bash]
----
nix build
----

=== Container (Stapeln)

[source,bash]
----
just stapeln-export    # Generates Containerfile
podman build -t statistease .
----

=== Manual Package

[source,bash]
----
just install --prefix=/usr/local
----

Files installed:

[cols="1,2"]
|===
| Path | Contents

| `$PREFIX/bin/`
| Executables

| `$PREFIX/share/statistease/`
| Data files, assets

| `$PREFIX/share/doc/statistease/`
| Documentation

| `$PREFIX/share/applications/`
| .desktop file (Linux, if GUI)

| `$PREFIX/share/man/man1/`
| Man pages
|===

== Configuration

Default config location: `$XDG_CONFIG_HOME/statistease/config.toml`

Fallback: `$HOME/.config/statistease/config.toml`

== Health Checks

[source,bash]
----
just doctor            # Full diagnostic
just run --version   # Version check
just run --selftest  # Built-in self-test
----

== Updating

[source,bash]
----
git pull
just build-release
just install --prefix=/usr/local
----

Or via OPSM: `opsm update statistease`

== Multi-Instance Deployment

For deploying multiple instances (e.g., different users or tenants):

[source,bash]
----
just install --prefix=/opt/statistease-instance1 --config=/etc/statistease/instance1.toml
just install --prefix=/opt/statistease-instance2 --config=/etc/statistease/instance2.toml
----

Each instance has isolated config, data, and logs.
